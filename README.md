# Лабораторная работа №2: Разработка лексического анализатора (сканера)

## Название и цель работы
**Тема:** Форматирование числа с плавающей точкой в научную нотацию на языке Python

**Цель работы:** Изучить назначение и принципы работы лексического анализатора в структуре компилятора. Спроектировать алгоритм (диаграмму состояний) и выполнить программную реализацию сканера для выделения лексем из входного текста, представляющих числа в различных форматах, с последующим преобразованием в научную нотацию.

## Постановка задачи
Разработать лексический анализатор (сканер), который:
- Принимает на вход строку с текстом программы
- Выделяет все допустимые лексемы, включая числа в различных форматах
- Классифицирует лексемы по типам
- Обнаруживает и сообщает об ошибках с указанием позиции
- Поддерживает многострочный ввод
- Интегрируется с графическим интерфейсом из лабораторной работы №1

## Вариант задания

### Описание работы автомата:

Конечный автомат предназначен для распознавания чисел с плавающей точкой в научной нотации:

- **Состояние 0 (НАЧАЛО)**: Начальное состояние, ожидание первого символа
- **Состояние 1 (Целая часть)**: Чтение целой части числа
- **Состояние 2 (Точка без целой части)**: Обнаружена десятичная точка без предшествующей целой части
- **Состояние 3 (Точка с целой частью)**: Обнаружена десятичная точка после целой части
- **Состояние 4 (Экспонента)**: Обнаружен символ 'E' или 'e'
- **Состояние 5 (Знак числа)**: Обработка знака числа (+ или -)
- **Состояние 6 (Дробная часть)**: Чтение дробной части
- **Состояние 7 (Знак экспоненты)**: Обработка знака экспоненты
- **Состояние 8 (Показатель степени)**: Чтение показателя степени

Автомат переходит в состояние **ДОПУСТИТЬ** при успешном завершении числа и в состояние **ОШИБКА** при обнаружении недопустимого символа.
